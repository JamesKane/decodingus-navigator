# qpAdm RIGHT populations — the outgroups the target's allele-sharing is measured against.
# One population code per line (blank lines and #-comments ignored). Appended to the panel axis
# after the left/source populations. This is the one expertise-driven artifact of the method
# (docs/design/ancient-ancestry-rebuild.md §7.4, §7.10).
#
# CRITICAL (learned the hard way — §7.10): the right set MUST come from the SAME genotype callset as
# the sources (the AADR 1240k EIGENSTRAT), NOT from a separate modern panel. Our first attempt sourced
# these from 1000G/SGDP shotgun while the sources were AADR capture, which ran a capture-vs-shotgun
# batch line straight through the f4 matrix and produced wildly infeasible weights (WHG −68%). Using
# AADR-native outgroups (below) fixed most of it (WHG −68% → −8%, model p 0.16 → 0.87).
#
# Requirements: differentially related to the three sources (so the f4 matrix has rank), NOT downstream
# of the mixture, and — for a frequency-based fit — DENSE ENOUGH that no single genome's idiosyncratic
# allele-sharing dominates. Single-genome anchors (Ust'-Ishim n=1) bias the fit, so the deep
# basal-Eurasian / ANE anchors are POOLED into denser groups (UPEuro, ANE) via aadr_component_map.tsv.
#
# All labels below are AADR Group-ID pools defined in aadr_component_map.tsv (present-day continental
# references + pooled deep anchors), so they share the sources' callset.
#
# code        lineage / role                                               approx n
Han         # Han Chinese (East Asian)                                       ~46
Papuan      # New Guinea (Oceanian) — deep non-African split                 ~32
Yoruba      # West African                                                    ~24
