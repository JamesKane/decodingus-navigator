# qpAdm RIGHT populations — the outgroups the target's allele-sharing is measured against.
# One population code per line (blank lines and #-comments ignored). Appended to the panel axis
# after the left/source populations. This is the one expertise-driven artifact of the method
# (docs/design/ancient-ancestry-rebuild.md §7.4): outgroups must be *differentially* related to the
# three sources (so the f4 matrix has rank) and must NOT be downstream of the mixture (no gene flow
# FROM the sources into an outgroup after the sources diverged).
#
# Sourced from the modern reference panels already extracted at the CHM13 panel sites
# (1000G = PRJEB31736/PRJEB36890, SGDP = PRJEB9586). Strong 1000G anchors pin the major continental
# axes at n≈100; the thin SGDP deep lineages supply the ANE / Oceanian / Andamanese differential
# relatedness that actually separates Steppe / ANF / WHG. n≈2–3 is normal for qpAdm outgroups (the
# textbook Lazaridis rightpops use Mbuti/Papuan/Onge/Karitiana at single-digit n); the f4 block
# jackknife accounts for the frequency noise. They get the lower --outgroup-min-called floor.
